# docs/ — master planning & specification documents

These are the source-of-truth planning documents the pre-registration is built on.
`protocol/preregistration.md` cites them by name ("spec v6.3", "plan v4.3"); they are
included here so the public repo is self-contained.

| File | What it is |
|---|---|
| `a1-requirements-and-alternatives-v6.3.md` | Requirements & alternatives specification (v6.3) — the design spec every freeze-list value derives from. |
| `A1-claude-science-execution-plan-v4.3.md` | Session-by-session execution plan (v4.3), built on spec v6.3. |

**These are planning documents, not frozen protocol.** They commit to `main` untagged and
evolve across versions; only `protocol/` material (the pre-registration) carries freeze tags.
Earlier versions (spec v6.1, plan v4.1, compute plan v1.0) live in the artifact store as
history and are intentionally not committed here — the repo tracks the current source of truth.
